Pré-formatação e códigos

com as tags <pre> e <code>, podemos ter uma interação interessante, com a utilização dessas tags conseguiremos digitar um código de python, javascript ou qualquer outra linguagem de programação em HTML sem ter falhas na sua formatção. Exemplos:(sem formatação) num = int(input('Digite um número: ')) if num %2 == 0 print(f'O número {num} é par!) else printf('O número {num} é ímpar!') print('FIM DO PROGRAMA')


Como podem perceber o texto está sem espaçamento e pulos de linha nenhum, devido a falta de formatação, agora vou fazer com o mesmo código porém com o uso das tags <pre> e <code>

                
    num = int(input('Digite um número: '))
    if num %2 == 0
        print(f'O número {num} é par!)
    else 
        printf('O número {num} é ímpar!')
    print('FIM DO PROGRAMA')

com a tag <code> nós deixamos as letras utilizadas monoespaçadas, isso é, todas vão "gastar" o mesmo espaço de pixels, e com a tag <pre> toda formatação utilizada no código-fonte, ou seja, cada linha e espaço utilizado no visual studio vai ser repassada para o site.
